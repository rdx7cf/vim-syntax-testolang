# TESTO-lang Vim Additions: Syntax Highlighting, Tab Insertions ...

## Installation
Copy all the directories into `~/.vim`.

**Optionally** add to your `.vimrc` the following lines:

`:filetype plugin on`

`:filetype plugin indent on`
